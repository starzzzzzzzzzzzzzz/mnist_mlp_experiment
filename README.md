# MNIST 手写数字识别实验

本项目旨在使用多层感知机 (MLP) 对 MNIST 数据集进行手写数字识别。

## 文件夹结构

```
mnist_mlp_experiment/
├── notebooks/          # 存放 Jupyter Notebooks
├── src/                # 存放 Python 脚本
├── data/               # 存放本地数据集 (如果需要)
├── models/             # 存放训练好的模型文件
└── README.md           # 项目说明文件
```
